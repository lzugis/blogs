课程介绍 ​

概览

Geoserver使用教程是一份保姆级的Geoserver操作指南，包括基础、进阶和扩展三个部分。在基础部分，将简单介绍Geoserver、Geoserver的安装、全局配置及如何使用Geoserver进行基本的地图发布和数据管理；在进阶部分，将介绍Geoserver的高级功能，如样式设置、切片原理、网格集以及地图切片；在扩展部分，将对PostGIS及其使用、OGC标准下地图服务、通过nginx进行集群。

一、基础教程 ​

课时课程内容
01.Geoserver介绍与安装 1.Geoserver介绍；
2.Geoserver环境与安装；
3.Geoserver主界面介绍；
02.全局配置 1.服务配置（WMS、WFS、WMTS）；
2.切片配置（Caching Defaults）；
03.工作空间、数据存储 1.工作空间管理；
2.数据存储管理；
    2.1 添加矢量数据（shp文件与文件夹、postgis、properties）；
    2.2 添加栅格数据（tif、镶嵌影像） ；
    2.3 添加数据存储插件（MySQL、Geopackage）；
    2.4 代理服务（WMS、WMTS）；
04.图层、图层组、图层预览 1.图层管理（发布图层）；
2.图层组管理；
3.图层预览（默认、WMS、WFS）；
